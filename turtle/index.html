Test Suite

New Version Available

The RDF Test Curation Community Group is maintaining updates to important test suites, including for Turtle. The new tests are available at http://w3c.github.io/rdf-tests/turtle/ along with an updated Implementation Report.

This directory contains one of the W3C RDF Working Group's test suites. Tests can be run from the web or by downloading either a tarball or zip file. See the LICENSE for the terms of use.

The manifest.ttl file lists all of the tests in the suite. See RDF 1.1 Test Cases for how to interpret this file.

Relative IRI Resolution: The home of the test suite is the URL of this page. Per RFC 3986 section 5.1.3, the base IRI for parsing each file is the retrieval IRI for that file. For example, the tests turtle-subm-01 and turtle-subm-27 require relative IRI resolution against a base of <http://www.w3.org/2013/TurtleTests/turtle-subm-01.ttl> and <http://www.w3.org/2013/TurtleTests/turtle-subm-27.ttl> respectively.
